# Uncover
## Description
Uncover est un jeu d'enquête, en incarnant un détective, il nous est donné comme mission de retrouver un collectionneur disparu.
Ce jeu a été développé dans le cadre du cours de développement de jeu vidéo 2d dispensé par Loïc Cattani.
![Image du Menu de Uncover](./assets/253-EM.JPG)
## Comment jouer?
Ce jeu se navigue à partir de la souris en mode point-and-click. Les zones cliquable transforme le curseur de la souris en croix.
## Dépendances
module kaplayjs
## Ressources
Photographies des arrière-plans faites par moi, génération des autres sprites à l'IA, parfois certaines retouches sur des images avec une application de retouche photo.
